# 📍 TripCheck

**TripCheck** は、旅行者が自分で旅程を作成し、ルートや費用を自動でチェックできるモバイルアプリです。  
旅行計画を直感的に作成し、複数の経路・交通費・宿泊費などを比較・保存できます。

---

## 🛠 使用技術スタック

- [Expo](https://expo.dev/)（React Native フレームワーク）
- TypeScript
- React Navigation
- Zustand（状態管理）
- AWS（バックエンド／インフラ予定）
- GitHub Actions（CI/CD予定）
